# Random Password Generator

A lightweight, interactive Python script that generates strong, randomized passwords based on user-defined lengths. The script utilizes a mix of uppercase and lowercase letters, digits, punctuation, and spaces to ensure high entropy and security.

## 🚀 Features

* **Custom Length:** Allows the user to specify exactly how long they want their password to be.
* **Diverse Character Pool:** Uses Python's built-in `string` module to automatically include:
  * Uppercase & Lowercase letters (`a-z`, `A-Z`)
  * Digits (`0-9`)
  * Punctuation & Special characters (`!@#$`, etc.)
  * Spaces

## 🛠️ Requirements

* **Python 3.x** (No external third-party libraries needed!)
