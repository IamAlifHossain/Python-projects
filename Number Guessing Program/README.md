# Number Guessing Game 🎮

A simple, interactive command-line number guessing game built with Python. The program randomly selects a number between 1 and 100, and challenges the player to guess it, providing real-time feedback along the way.

## Features
* **Dynamic Feedback:** Tells the player if their guess is "Too high" or "Too low".
* **Score Tracking:** Tracks and displays the total number of attempts it took to guess the correct number.
* **Randomization:** Uses Python's `random` module to ensure a unique secret number every time you play.
