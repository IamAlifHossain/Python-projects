# Voice-Enabled Countdown Timer

A sleek, lightweight Python-based countdown timer that utilizes a graphical user interface and audio cues to keep you on track. This project is built using Python 3.12.x and leverages the power of `pygame` for seamless audio playback.

---

## Necessary Accessories & Requirements

To run this project smoothly, ensure you have the following environment and dependencies set up:

### 1. IDE / Text Editor
* **Visual Studio Code (VS Code)**: Recommended for development. 

### 2. Python Version
* **Python 3.12.x**: This project is optimized for Python 3.12. Ensure your local environment or virtual environment is running a compatible version.

### 3. Required Modules & Libraries
* **Pygame**: Used for managing timer audio alerts and background sounds.
