# 🛒 Interactive Menu Order Program

A simple, interactive command-line interface (CLI) program written in Python that allows users to browse a food menu, add items to a digital shopping cart, and calculate the total bill dynamically.

---

## 🚀 Features

* **Dynamic Menu Display:** Neatly formats and aligns food items and prices using Python f-strings.
* **Interactive Shopping Cart:** Users can enter items one by one to add them to their cart.
* **Input Validation:** Only items that exist on the menu can be added. Case-insensitive exit command (`q` or `Q`).
* **Itemized Receipt:** Displays an indexed list of ordered items and the exact total bill formatted to two decimal places.
